OpenCV with C++

By solitary_crow

Sep 17 2024

Almost all examples on the web use Python as the language of choice for using opencv mainly because it's easier to prototype with Python. But Python leaves a lot at the table in terms of performance and other requirements for deploying opencv applications. Using C++ would allow you to understand how OpenCV works and how to customize it for your own needs. This book requires you to know basic modern C++ and Cmake as the build tool.
