Chapter 2

Compilers in practice

In this book we will go through some opensource commercial compilers that are used in the software industry today. Among them will be GCC, LLVM, MLIR. LLVM is the main focus of this book since it's language agnostic and provides advanced optimization and analysis. Beside LLVM we will also look into compiler implementation of functional programming languages.

What this book aims to be

This book is heavy on practical implementation rather than theoratical consideration. For that there are lots of other books. There is a lot to writing a compiler from frontend to the backend. Writing a compiler is not too hard but writing an optimizing compiler is a difficult task. There are already a lot of resource material on writing basic compilers like,

  • Crafting interpreters

Books on compilers

List of general compiler books

  • Compilers: Principals, Techniques and Tools by Alfred V. Aho et. al.

    This book is called the "dragon book" and is one of the most famous compiler book out there. I have seen people argue how relevent it is in modern times.

  • Modern compiler implementation in ML by Adrew W. Appel

    This is a really good book on practical compiler construction in ML functional programming language. There are C and Java versions too but ML would be a better alternative to them for compiler construction.

  • Engineering A compiler by Keith D. Cooper

    A modern compiler book but lots on theory. A better alternative to the Dragon book.

  • Modern Compiler Design by Dick Grune

    A modern alternative to the dragon book.

Advanced books on optimizing compiler construction

  • Advanced Compiler Design and Implementation by Steven Muchnick

    An extensive resource on algorithms used in modern optimizing compilers. A really good book for understanding industrial stregth optimizing compilers.

  • High Performance compilers for Parallel computing by Michael Wolf

    The book by Steven Muchnick mostly focuses on optimizing programs for uniprocessor computers. This book fills the gap and provides a comprehensive introduction to transformation and analysis for parallel computing.
